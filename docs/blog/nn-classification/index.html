神经网络分类任务的底层原理：从训练到推理

前言

很多人在调用 model.fit() 和 model.predict() 的时候，并不知道底层到底发生了什么。这篇文章用最简单的二分类任务，从零开始推导神经网络的训练和推理过程。读完你就能理解：权重到底是怎么学会分类的。

我们只用到最基础的数学——加权求和、链式法则、sigmoid——没有框架，没有黑盒。

1. 分类任务是什么

假设桌子上放了一堆苹果和橘子，你闭着眼睛伸手摸一个水果，只能感知两个特征：

  • 重量（克）
  • 硬度（捏一下的手感）

你的目标：只靠这两个数字，判断它是苹果还是橘子。

这就是一个二分类任务。输入是一个二维向量 x = [重量, 硬度]，输出是 000（苹果）或 111（橘子）。

把数据画在二维平面上，数据大概是这样的：

样本重量硬度类别
11806.5苹果 (0)
22204.0橘子 (1)
31607.0苹果 (0)
42403.5橘子 (1)

如果把这些点画在平面上，苹果和橘子大概会分成两坨。神经网络要做的事就是画一条线（或曲线），把这两坨分开。

2. 一个最简单的神经网络

我们用一个只有输入层和输出层的网络（也叫逻辑回归的神经网络形式）：

数学表达式分两步：

第一步：加权求和

z=w1⋅x1+w2⋅x2+bz = w_1 \cdot x_1 + w_2 \cdot x_2 + bz=w1​⋅x1​+w2​⋅x2​+b

其中 w1w_1w1​、w2w_2w2​ 是权重（weight），bbb 是偏置（bias）。

第二步：激活函数

y^=σ(z)=11+e−z\hat{y} = \sigma(z) = \frac{1}{1 + e^{-z}}y^​=σ(z)=1+e−z1​

sigmoid 函数将任意实数 zzz 压缩到 (0,1)(0, 1)(0,1) 区间。为什么这样做？因为我们需要输出一个概率——预测为橘子（类别 1）的概率。

Sigmoid 的形状像 S：

  • 当 z→+∞z \to +\inftyz→+∞，σ(z)→1\sigma(z) \to 1σ(z)→1（非常确定是橘子）
  • 当 z→−∞z \to -\inftyz→−∞，σ(z)→0\sigma(z) \to 0σ(z)→0（非常确定是苹果）
  • 当 z=0z = 0z=0，σ(z)=0.5\sigma(z) = 0.5σ(z)=0.5（不确定）

这就构成了整个**推理（前向传播）**过程：

输入 x  ⟶  加权求和 z  ⟶  sigmoid  ⟶  输出概率 y^\text{输入 } x \;\longrightarrow\; \text{加权求和 } z \;\longrightarrow\; \text{sigmoid} \;\longrightarrow\; \text{输出概率 } \hat{y}输入 x⟶加权求和 z⟶sigmoid⟶输出概率 y^​

3. 推理过程详解

假设我们已经训练好了网络，拿到了权重 w1=0.5w_1 = 0.5w1​=0.5、w2=−0.3w_2 = -0.3w2​=−0.3、偏置 b=0.1b = 0.1b=0.1。

现在来了一个新样本：重量 200g，硬度 5.0。

第一步：计算 z

z=0.5×200+(−0.3)×5.0+0.1z = 0.5 \times 200 + (-0.3) \times 5.0 + 0.1z=0.5×200+(−0.3)×5.0+0.1 z=100−1.5+0.1=98.6z = 100 - 1.5 + 0.1 = 98.6z=100−1.5+0.1=98.6

第二步：过 sigmoid

y^=σ(98.6)=11+e−98.6≈1.0\hat{y} = \sigma(98.6) = \frac{1}{1 + e^{-98.6}} \approx 1.0y^​=σ(98.6)=1+e−98.61​≈1.0

y^≈1.0\hat{y} \approx 1.0y^​≈1.0，远超 0.5 的阈值。模型非常确定这是橘子。

再来一个：重量 150g，硬度 7.0。

z=0.5×150+(−0.3)×7.0+0.1z = 0.5 \times 150 + (-0.3) \times 7.0 + 0.1z=0.5×150+(−0.3)×7.0+0.1 z=75−2.1+0.1=73.0z = 75 - 2.1 + 0.1 = 73.0z=75−2.1+0.1=73.0 y^=σ(73.0)≈1.0\hat{y} = \sigma(73.0) \approx 1.0y^​=σ(73.0)≈1.0 — 仍然是橘子。

嗯……这个结果不太对，150g 轻、7.0 硬，应该是苹果才对。这说明权重还是随机的——我们还没开始训练。

4. 训练：让网络学会分类

训练的目的就是找到一组 w1w_1w1​、w2w_2w2​、bbb，使预测结果尽可能接近真实标签。

训练分为三步，循环执行：

前向传播  ⟶  计算损失  ⟶  反向传播（更新权重）  ⟶  重复\text{前向传播} \;\longrightarrow\; \text{计算损失} \;\longrightarrow\; \text{反向传播（更新权重）} \;\longrightarrow\; \text{重复}前向传播⟶计算损失⟶反向传播（更新权重）⟶重复

4.1 初始化

开始时，权重是随机初始化的。假设 w1=0.5w_1 = 0.5w1​=0.5、w2=0.3w_2 = 0.3w2​=0.3、b=0.0b = 0.0b=0.0。

4.2 前向传播

拿样本 1：x=[180,6.5]x = [180, 6.5]x=[180,6.5]，真实标签 y=0y = 0y=0（苹果）。

z=0.5×180+0.3×6.5+0.0=91.95z = 0.5 \times 180 + 0.3 \times 6.5 + 0.0 = 91.95z=0.5×180+0.3×6.5+0.0=91.95 y^=σ(91.95)≈1.0\hat{y} = \sigma(91.95) \approx 1.0y^​=σ(91.95)≈1.0

模型说 y^≈1.0\hat{y} \approx 1.0y^​≈1.0，但真实标签 y=0y = 0y=0。错得非常离谱。

4.3 损失函数：量化「错得多离谱」

二分类最常用的损失函数是交叉熵（Cross-Entropy）：

L=−[ y⋅ln⁡(y^)+(1−y)⋅ln⁡(1−y^) ]L = -[\,y \cdot \ln(\hat{y}) + (1-y) \cdot \ln(1-\hat{y})\,]L=−[y⋅ln(y^​)+(1−y)⋅ln(1−y^​)]

套入我们的数据：y=0y=0y=0，y^≈1.0\hat{y} \approx 1.0y^​≈1.0。

L=−[ 0⋅ln⁡(1.0)+1⋅ln⁡(0.000…) ]L = -[\,0 \cdot \ln(1.0) + 1 \cdot \ln(0.000\ldots)\,]L=−[0⋅ln(1.0)+1⋅ln(0.000…)] ln⁡(0.000…)→−∞\ln(0.000\ldots) \to -\inftyln(0.000…)→−∞

所以 LLL 是一个非常大的正数，说明模型错得离谱。这很合理——明明标签是苹果，模型却说 99.999% 是橘子。

反过来，如果 y^\hat{y}y^​ 接近 yyy，损失就小。这是一张直观的理解表：

yyyy^\hat{y}y^​损失解读
0（苹果）0.1小模型正确，很确信是苹果 ✓
0（苹果）0.9大模型错误，几乎确信是橘子 ✗
1（橘子）0.8小模型正确，比较确信是橘子 ✓
1（橘子）0.2大模型错误 ✗

4.4 反向传播：修正错误

现在最关键的步骤来了——我们需要调整 w1w_1w1​、w2w_2w2​、bbb 来降低损失。

核心思想：梯度下降

想象你站在山顶，大雾弥漫，你想到山谷最低点。你看不到全景，但能感知脚下地面的坡度。于是你每次往最陡的下坡方向走一小步。重复足够多次，你就能到达谷底。

损失函数就是那座山。梯度告诉你哪个方向的损失下降最快。

链式法则推导

我们要计算 ∂L∂w1\frac{\partial L}{\partial w_1}∂w1​∂L​、∂L∂w2\frac{\partial L}{\partial w_2}∂w2​∂L​、∂L∂b\frac{\partial L}{\partial b}∂b∂L​。

拆成三个计算步骤（链式法则）：

L  ⟶  ∂L∂y^  ⟶  ∂L∂y^⋅∂y^∂z  ⟶  ∂L∂y^⋅∂y^∂z⋅∂z∂w1L \;\longrightarrow\; \frac{\partial L}{\partial \hat{y}} \;\longrightarrow\; \frac{\partial L}{\partial \hat{y}} \cdot \frac{\partial \hat{y}}{\partial z} \;\longrightarrow\; \frac{\partial L}{\partial \hat{y}} \cdot \frac{\partial \hat{y}}{\partial z} \cdot \frac{\partial z}{\partial w_1}L⟶∂y^​∂L​⟶∂y^​∂L​⋅∂z∂y^​​⟶∂y^​∂L​⋅∂z∂y^​​⋅∂w1​∂z​

第一步：交叉熵的梯度

∂L∂y^=−yy^+1−y1−y^\frac{\partial L}{\partial \hat{y}} = -\frac{y}{\hat{y}} + \frac{1-y}{1-\hat{y}}∂y^​∂L​=−y^​y​+1−y^​1−y​

第二步：sigmoid 的梯度

sigmoid 有一个优美的性质：

σ′(z)=σ(z)⋅(1−σ(z))\sigma'(z) = \sigma(z) \cdot (1 - \sigma(z))σ′(z)=σ(z)⋅(1−σ(z))

所以：

∂y^∂z=y^⋅(1−y^)\frac{\partial \hat{y}}{\partial z} = \hat{y} \cdot (1 - \hat{y})∂z∂y^​​=y^​⋅(1−y^​)

第三步：合并

∂L∂z=∂L∂y^⋅∂y^∂z\frac{\partial L}{\partial z} = \frac{\partial L}{\partial \hat{y}} \cdot \frac{\partial \hat{y}}{\partial z}∂z∂L​=∂y^​∂L​⋅∂z∂y^​​

代入化简后得到一个极其简洁的结果：

∂L∂z=y^−y\frac{\partial L}{\partial z} = \hat{y} - y∂z∂L​=y^​−y

就一行！预测值减去真实值。这意味着：

  • 如果 y^=0.9\hat{y} = 0.9y^​=0.9，y=0y = 0y=0 → 梯度 = 0.90.90.9（正数，需要减小 y^\hat{y}y^​）
  • 如果 y^=0.1\hat{y} = 0.1y^​=0.1，y=1y = 1y=1 → 梯度 = −0.9-0.9−0.9（负数，需要增大 y^\hat{y}y^​）
  • 如果 y^≈y\hat{y} \approx yy^​≈y → 梯度 ≈0\approx 0≈0（接近最优，不调整）

最后一步：求各参数的梯度

∂L∂w1=∂L∂z⋅x1=(y^−y)⋅x1\frac{\partial L}{\partial w_1} = \frac{\partial L}{\partial z} \cdot x_1 = (\hat{y} - y) \cdot x_1∂w1​∂L​=∂z∂L​⋅x1​=(y^​−y)⋅x1​

∂L∂w2=∂L∂z⋅x2=(y^−y)⋅x2\frac{\partial L}{\partial w_2} = \frac{\partial L}{\partial z} \cdot x_2 = (\hat{y} - y) \cdot x_2∂w2​∂L​=∂z∂L​⋅x2​=(y^​−y)⋅x2​

∂L∂b=∂L∂z=y^−y\frac{\partial L}{\partial b} = \frac{\partial L}{\partial z} = \hat{y} - y∂b∂L​=∂z∂L​=y^​−y

4.5 更新权重

拿到梯度后，沿梯度的反方向走一小步（梯度指向损失上升的方向，我们要下降）：

w1:=w1−α⋅(y^−y)⋅x1w_1 := w_1 - \alpha \cdot (\hat{y} - y) \cdot x_1w1​:=w1​−α⋅(y^​−y)⋅x1​ w2:=w2−α⋅(y^−y)⋅x2w_2 := w_2 - \alpha \cdot (\hat{y} - y) \cdot x_2w2​:=w2​−α⋅(y^​−y)⋅x2​ b:=b−α⋅(y^−y)b := b - \alpha \cdot (\hat{y} - y)b:=b−α⋅(y^​−y)

其中 α\alphaα 是学习率（learning rate），控制每步走多大。太小收敛慢，太大可能跳过最优点。

关键直觉：

  • 当模型错判为 0（实际是 1），梯度是负数 → 加回权重，增大 y^\hat{y}y^​

这就是神经网络的本质：犯错 → 算梯度 → 往反方向修正 → 下次错得少一点。重复几千次，权重自然学会了如何区分。

特征缩放的重要性

回顾一下数据：重量在 150-250 范围，硬度在 3-8 范围。当前权重下 zzz 的值被重量项主导（0.15×180=270.15 \times 180 = 270.15×180=27），硬度的贡献微不足道。

更严重的是，sigmoid 在 ∣z∣>5|z| > 5∣z∣>5 时就饱和了——梯度几乎为零，学不动了。这叫做梯度消失。

解决方案：归一化

把特征缩放到相似的量级，比如 0-1 区间：

xnorm=x−minmax−minx_{norm} = \frac{x - min}{max - min}xnorm​=max−minx−min​

这预测为苹果？不对……

手动检查一下：样本 D 是橘子（y=1y=1y=1），归一化后 x=[1.0,0.0]x=[1.0, 0.0]x=[1.0,0.0]。硬度为 0，重量最大，按理说「重 + 不硬 = 橘子」应该对，但模型却说苹果。

问题在于这个数据集线性不可分——苹果不是全部在一根线的一边。看来只用 2 个特征 + 线性分类器不够。

这恰恰说明了一个要点：单层神经网络本质上是线性分类器，它只能画一条直线（或高维空间中的超平面）。对于复杂数据，你需要隐藏层。

引入隐藏层

加一个隐藏层，网络变成这样：

第一层将输入非线性地映射到「隐藏空间」——在隐藏空间里，原来线性不可分的数据变得可分。第二层在隐藏空间里画一条直线，轻松分开。

这就是神经网络的核心魔法：堆叠多层非线性变换，将不可分变得可分。

6. 推理：最终形态

训练完毕的神经网络推理过程非常简单——就是一遍前向传播，没有任何反向步骤：

  1. 拿到新样本的特征向量 xxx
  2. 逐层往前算：x→h(1)→h(2)→⋯→y^x \to h^{(1)} \to h^{(2)} \to \dots \to \hat{y}x→h(1)→h(2)→⋯→y^​
  3. 如果是二分类，y^>0.5\hat{y} > 0.5y^​>0.5 判为类别 1，否则类别 0
  4. 如果是多分类，最后一层用 softmax，选概率最大的类别

推理不需要梯度、不需要反向传播、不需要损失函数。那些只在训练阶段使用。

7. 总结

关键概念速查表：

概念直白解释
权重网络判断时，每个特征的「重要程度」
偏置当所有特征都是 0 时，网络的「基础倾向」
激活函数引入非线性，否则叠多少层都只是一次线性变换
前向传播拿特征算出预测结果
损失函数我们定义的一个数字，告诉网络「你错得多离谱」
梯度指向损失上升最快的方向——网络往反方向走
学习率每步走多大，太小学得慢，太大跳过头
反向传播用链式法则把误差一层一层传回去，每层各算各的梯度
归一化让特征在相似量级，否则梯度爆炸/消失
隐藏层当数据线性不可分时，用非线性变换找新的表示空间

神经网络不是什么魔法——它就是反复犯错、计算梯度、修正自己的数学机器。理解了这篇文章里的每个公式，你就真正懂了深度学习的第一块基石。
